🚀 About GitHub Time Tracker

GitHub Time Tracker is a lightweight browser extension that integrates time tracking directly into your GitHub workflow. With a sleek Start/Stop Timer button on every Issue page and a new calendar view, track and manage your task time effortlessly.

🚀 Key Features

🔘 Start/Stop Timer

  • Custom timer button on each Issue.
  • One-click start and stop.
  • Real-time live counter.

📅 Calendar View

  • New in v1.0.2: Visualize time by day.
  • Select any day to view tasks.
  • Search and see daily totals.

📋 Time History & Search

  • View time history in popup.
  • Sort issues by date.
  • Search by title or URL.

🔁 Seamless Task Switching

  • Auto-stops previous timer.
  • Prevents overlaps, saves progress.

🔐 GitHub Token Integration

  • Optional token for commenting.
  • Posts time to issues.

🧠 Smart Persistence

  • Local storage via Chrome API.
  • Persists across restarts.
  • Saves progress on crash.

🌟 What's Next?

We're planning a major refactoring to improve performance and introduce customizable features like time format options, UI themes, and advanced analytics. Stay tuned for a more tailored and powerful tracking experience!

🧩 Why Choose GitHub Time Tracker?

  • Native GitHub Integration: Works within GitHub, no context switching.
  • No External Dependencies: No accounts or third-party APIs.
  • Offline Capability: Fully functional offline, local storage.
  • Privacy-Conscious: No analytics or data sharing.
  • Lightweight & Instant: Install and use immediately.

📦 Who Is This For?

Whether you're a solo developer, freelancer, or part of a team, GitHub Time Tracker helps you:

  • Stay accountable for your time.
  • Understand time distribution across tasks.

All without interrupting your workflow.
