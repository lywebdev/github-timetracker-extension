Privacy Policy

Last updated: May 11, 2025

GitHub Time Tracker is a browser extension that helps users track time spent working on GitHub issues. We respect your privacy and take data protection seriously. This extension is designed to operate entirely on your device without sending any personal data externally.

Data Collected

  • Time tracking logs
  • Active task and user preferences
  • (Optional) GitHub Personal Access Token, stored locally for comment posting

All data is stored locally in your browser using the Chrome Storage API and is never transmitted to any external servers.

Data Usage

Collected data is used solely to:

  • Enable and persist time tracking functionality
  • Display historical data and active tasks in the popup
  • (Optionally) Post time tracking comments to GitHub using your personal token

The extension does not include analytics, ads, telemetry, or data sharing with third parties.

Permissions

  • storage – to store logs, settings, and active task data
  • activeTab – to interact with the current GitHub tab when popup is opened
  • https://github.com/*/*/issues/* – to inject a timer button on GitHub Issue pages

Security

Your GitHub token and time tracking history remain entirely on your device. We do not transmit any data to external servers or services.
